UV Cockpit Web App Suite

UV Tracks service provides access to the mission plans and reported states saved to the vehicle shadow by UV Hub server of SPL Global Telemetry System.

Refer to UV Tracks REST API documentation for instructions about how to use the service from applications.

UV Tracks 3D Demo and UV Tracks 2D Demo are web applications that demonstrates use of UV Tracks web service for tracking unmanned aerial vehicles on a 3D and 2D maps.
