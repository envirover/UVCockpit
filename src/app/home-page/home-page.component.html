UV Cockpit

UV Cockpit currently includes UV Tracks 3D Demo and UV Tracks 2D Demo web applications that demonstrate use of UV Tracks web service for tracking of unmanned vehicles on a 3D and 2D maps.

UV Tracks service provides access to the mission plans and reported states saved to the vehicle shadow by UV Hub server of SPL Global Telemetry System.

Refer to UV Tracks REST API v2 documentation for instructions on how to use the service from applications.
